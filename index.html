What is CSS?

  1. CSS stands for Cascading style sheets.
  2. CSS describe how html elements are to be displayed on screen, paper or in other media.
  3. Css is not necessary Css saves alot of work.It can control the layout of multiple web pages.
  4. External style are stored in CSS file.

CSS can be added to HTML documents in 3 ways.

  • Inline-by using the style attribute inside html elements.
  • Internal-by using the style attribute inside HTML elements.
  • External- by using a link element to link external CSS file.
  • The most common ways to add CSS is to keep the style in external CSS fileHowever in this tutorial we will use inline & internal styles. because this easier to demonstrate & easier for you to try it yourself.

When comparing CSS class vs ID, the difference is that CSS class apllies a style to multiple elements. ID on the other hand, applies a style to one unique element. ID is also special in that you can use a special URL to link directly & its used by Javascript.

In CSS, selector are used to target a specify element or range of elements on a web page. One on element has been targeted style or set of style can be applied to the elements.

Ther is a wide range of selector available two of the most commonly used used to target elements to which a style should be applied.
